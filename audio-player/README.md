# 音频播放器

支持断点播放和批量上传的网页音频播放器。

## 功能特性

- 邮箱注册/登录
- 批量上传音频（支持100个文件，单文件最大1GB）
- 断点播放（自动保存进度）
- 响应式设计（支持手机访问）
- 支持多种音频格式（MP3, WAV, OGG, FLAC, M4A等）

## 启动方式

```bash
# 直接启动
node server.js
```
