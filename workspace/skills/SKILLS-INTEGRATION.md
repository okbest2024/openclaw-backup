## 🧬 跨领域元模式

从 100+ 技能中，我识别出以下**跨领域元模式**：

### 模式 1：分层架构
```
感知层 → 理解层 → 决策层 → 执行层 → 反馈层
```
- 知识获取技能 = 感知
- 分析技能 = 理解
- 规划技能 = 决策
- 自动化技能 = 执行
- 自我改进技能 = 反馈

### 模式 2：闭环学习
```
行动 → 结果 → 反思 → 固化 → 新行动
```
- self-improving：记录纠正，3 次固化
- agent-self-reflection：每小时反思
- executing-plans：执行后回顾

### 模式 3：能力组合
```
单一技能 → 技能链 → 工作流 → 系统
```
- 搜索 + 阅读 + 总结 = 研究能力
- 监控 + 分析 + 通知 = 运维能力
- 设计 + 生成 + 发布 = 创作能力

### 模式 4：容错设计
```
假设失败 → 检测失败 → 恢复机制 → 记录教训
```
- n8n-workflow-automation：错误处理、重试
- auto-updater：备份、回滚
- backtest-expert：压力测试、鲁棒性检查
