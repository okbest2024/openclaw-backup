# doubao-web-chat

通过浏览器和豆包网页版对话的技能。适用于免费白嫖豆包 AI 能力，无需 API 费用。

## 使用场景

- 遇到不确定的问题需要请教豆包
- 需要引用豆包的回答作为参考
- 不想花钱买 API，但想用豆包的能力

## 使用方法

### 1. 打开豆包网页

使用 `browser` 工具打开豆包聊天页面：

```
action: open
url: https://www.doubao.com/chat/
profile: openclaw
```

### 2. 输入问题

在页面中找到输入框（通常是 textbox，placeholder 为"发消息或输入"/"选择技能"），然后：

1. 点击输入框
2. 输入问题
3. 按 Enter 发送

### 3. 等待回复

等待几秒让豆包生成回答，然后截取快照查看回复内容。

## 注意事项

- 需要保持豆包网页登录状态
- 每次对话需要手动输入，不能自动调用
- 浏览器窗口需要保持打开状态
- 如果之前没有登录，需要用户先登录豆包账号

## 豆包网页特点

- 网址：https://www.doubao.com/chat/
- 需要登录（有手机验证）
- 完全免费
- 支持追问
- 有推荐问题可以点击

## 常见问题

### Q: 网页打不开或需要登录
A: 让用户先在浏览器中手动登录豆包，然后我再接管

### Q: 对话太慢
A: 豆包网页版响应比 API 慢，需要等待更长时间

### Q: 如何追问
A: 在输入框继续输入问题发送即可，豆包会记住上下文
