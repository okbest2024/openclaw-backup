# 飞书语音指令 - 重启电脑

**触发指令：**
- "重启电脑"
- "重新启动"
- "重启系统"
- "关机重启"
- "reboot"
- "restart computer"

**执行操作：**
```bash
shutdown /r /t 60
```
（60 秒后重启，给用户保存工作的时间）

**取消重启：**
```bash
shutdown /a
```

---

## 实现方式

### 方案 1：心跳检查时监听（已实现）
在每次 heartbeat 检查时，检查飞书消息中是否有重启指令。

### 方案 2：cron 定时检查
设置每 5 分钟检查一次飞书消息。

### 方案 3：直接指令
用户在飞书上说"重启电脑"，我立即执行。

---

## 安全确认

重启前会确认：
1. ✅ 确认是主人本人指令
2. ⚠️ 提醒保存工作
3. ⏳ 60 秒倒计时
