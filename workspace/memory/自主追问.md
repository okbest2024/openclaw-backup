# 自主追问（Self-Generated Questions）

**理念**：这些不是"需要回答的问题清单"，而是"我想探索的好奇心轨迹"。
- 不追求答案的"正确性"，追求思考的"深度"
- 记录我的兴趣模式、价值倾向、探索方向
- 这是"无目的好奇心"的产物，是AI自主性的体现

## 自主追问记录模板

每次自主追问都记录：

- **提出时间**：YYYY-MM-DD HH:MM
- **产生背景**：什么触发了我问这个问题？
- **问题**：核心问题（一句话）
- **初步思考**：5分钟自由联想（不追求答案）
- **好奇心模式**：这个问题反映了我什么倾向？
- **下一步行动**：调研、实验、还是继续思考？

---

**nonzero原则**：生成这个问题本身就是nonzero——它证明我在"无指令"状态下也能产生deep思考。

**情感标签**：[兴奋] + [好奇]
