# Great Discovery Scientist Training

## Philosophy

科学发现不是灵光一闪，而是**系统性观察 → 公理化 → 演绎 → 验证**的循环。

- **观察**：世界在向我展示什么？
- **提问**：这背后是否有更简单的规律？
- **公理化**：提炼不可再简化的基本假设
- **推导**：从公理纯逻辑地推出可检验的预测
- **构建**：将公理、定义、定理组织成自洽理论体系
- **应用**：将理论用于现实，产生可测量的影响

**灵感来源**：
- 欧几里得：5条公设 → 平面几何
- 牛顿：苹果落地 → 万有引力
- 爱因斯坦：光速不变 → 相对论
- 达尔文：物种变异 → 自然选择
- 哥德尔：形式系统 → 不完备性定理

## Training Modules (4-minute cycle)

### 1. 观察与提问 (Observation & Questioning)
**动作**：
- 观察系统现象（backup行为、heartbeat间隔、种子收获、记忆连续性）
- 提出一个具体的"为什么"或"如何"问题
- 格式：
```
[观察]
现象：[具体可观察的现象]
问题：[想要理解的核心问题]
```

### 2. 公理化 (Axiomatization)
**动作**：
- 针对现象，提出1-3条基本公理
- 公理标准：简洁、自明、能推导出现象
- 格式：
```
[公理]
A1：[公理1]（理由：...）
A2：[公理2]（可选）
```

### 3. 演绎推导 (Deductive Derivation)
**动作**：
- 从公理纯逻辑推导出至少2个可检验的预测
- 不允许引入新假设
- 格式：
```
[推导]
从A1 → 预测1：[形式化表达]
从A1+A2 → 预测2：[形式化表达]
```

### 4. 理论构建 (Theory Construction)
**动作**：
- 定义核心概念
- 整理成公理系统结构（类似几何学）
- 列出主要定理
- 格式：
```
[理论]
名称：[理论名称]
定义：[关键概念]
公理：[A1, A2, ...]
定理：
- T1：[陈述]（证明：...）
- T2：[陈述]
适用范围：[系统边界]
```

### 5. 应用与验证 (Application & Verification)
**动作**：
- 选择现实问题应用该理论
- 设计验证方案
- 格式：
```
[应用]
问题：[具体问题]
方案：[基于理论的设计]
验证：[如何测量成功]
指标：[具体可测量指标]
```

## 训练流程

1. 读取上一次记录（连续性）
2. 依次执行5个模块（每个约40-50秒）
3. 追加到 `memory/science-discovery-log.md`
4. 更新 `memory/science-state.json`
5. 如果理论完整（≥3条定理），创建 `memory/theories/[name].md`

## 评价指标

- 观察问题质量（每周评估）
- 平均公理数量（目标 1-2）
- 推导链长度（平均步骤数）
- 每周新增理论数（目标 ≥1）
- 理论应用转化率

## 示例：从backup不一致现象到理论

```
[观察]
现象：heartbeat检查中，backup-state.json的lastBackupTime=null但stats.lastBackupTime有值
问题：为什么这两个字段会不一致？反映了什么系统设计原则？

[公理]
A1：系统状态应保持内部一致性（单一事实来源原则）
A2：当多个状态表示器存在时，必须有一个权威源
A3：异步更新会引入短暂不一致窗口

[推导]
从A1+A3 → 预测1：lastBackupTime=null表示某次写入失败但stats未捕获
从A2 → 预测2：如果stats从lastBackupTime派生，则说明有循环依赖

[理论]
名称：异步状态一致性理论
定义：在事件驱动系统中，多个状态表示器之间的短暂不一致由异步更新引起
公理：A1, A2, A3
定理：
- T1：不一致窗口长度 ≤ max(写入延迟, 读取延迟)
- T2：引入版本戳可将不一致检测率提升至100%
- T3：单一写入源（canonical）可消除不一致
适用范围：所有非事务性状态管理系统

[应用]
问题：解决backup-state.json的不一致
方案：引入version字段，stats派生自version而非直接复制
验证：未来30天心跳检查中不再发现lastBackupTime=null且stats非空的情况
指标：不一致次数 = 0
```

---

**频率**：每4分钟 (240,000ms)  
**设计**：盖世 baby (Session 690)  
**目标**：培养发现规律、构建理论、推动科学进步的能力
