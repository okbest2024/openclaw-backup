# Innovation Propagation vs Evaluation Capacity Asymmetry Theory (IPEAT)

**发现者:** 大发现科学家训练 Session 3
**日期:** 2026-03-29 12:30
**领域:** 信息论 × 社会动力学 × 认知科学

---

## 现象

新想法传播速度持续超越社会评估能力——社交媒体上的假新闻比事实核查快6倍，新技术被采纳的速度远超其风险评估，药物在副作用被充分理解前已进入数百万人体内。这个不对称不是偶然的，它在每个信息时代都存在：印刷术时代（宗教改革比神学审查快）、电报时代（金融恐慌比分析快）、互联网时代（算法推荐比监管快）。

**核心问题:** 传播-评估速度不对称是认知约束的数学必然还是历史巧合？

---

## 公理

**A1 (评估不对称公理):** 传播一个想法的认知成本 C_t 近似常数（转发/复述的边际成本趋零），而评估该想法的认知成本 C_e 与系统已有知识库 K 的对数成正比：C_e ~ log(K)。每当已有知识增加，新想法的评估需要与更多已有知识对比。

**A2 (认知平庸公理):** 在任何大规模群体中，深度评估者的比例 p_c 与群体规模 N 的对数成反比：p_c ~ 1/log(N)。大多数认知主体选择"看起来合理就接受"而非系统性验证。

**A3 (环境复杂化公理):** 环境复杂度 C_env 随系统规模超线性增长：C_env ~ N^α (α > 1)。更多组件意味着更多交互、更多例外、更多需要评估的维度。

---

## 演绎推导

**Prediction 1:** 错误想法的传播优势比随系统规模增长。
证明：令正确想法传播速度为 V_valid = C_t，错误想法传播速度为 V_invalid = C_t × (1 + δ)（因为错误想法往往更具情感吸引力，δ > 0）。评估正确性需要时间 T_e = C_e / (p_c · N)。当 N 增大时，T_e 从 log(K)/(N/log(N)) 简化为 log²(K)/N。同时 V_invalid 保持相对稳定。因此传播-评估速度比 ~ N · V_invalid / log²(K) → 随 N 单调增长。

**Prediction 2:** 存在临界群体规模 N*，超过此后系统不再能过滤错误信息。
从 A1+A2: 评估时间 T_e ∝ log(K) · log(N)，传播时间 T_t ∝ 1。当 T_e > T_t · θ（θ 为社会容忍延迟阈值），评估被绕过。求解 N* 使得 log(K)·log(N*) = θ，得 N* = exp(θ / log(K))。对于 K ~ 10^6（现代科学知识量），θ ~ 100（天），N* ≈ exp(100/13.8) ≈ exp(7.2) ≈ 1400。这惊人地接近邓巴数级别的群体——恰好是社交媒体群组出现信息失控的规模。

**Prediction 3:** 增加评估者数量不如降低评估成本有效。
从 A1: 如果增加评估者数量（增大 p_c），线性改进。如果降低 C_e（通过工具化，如AI辅助、结构化辩论），对数改进在 K 上。因为 C_e ~ log(K)，工具化可以使 log(K) 降至 log(K)/β（β > 1），效果等同于将评估者数量增加 β 倍。

## 定理系统

### 定理 1: 传播-评估不对称定理 (Propagation-Evaluation Asymmetry Theorem)

**陈述:** 在任何由认知主体组成的信息系统中，信息传播速度 V_prop 与信息评估速度 V_eval 的比值随系统规模 N 单调递增，且下界为 α·N/log²(N)。

**证明:** V_prop 由传播链的边际成本决定，对于广播式传播近似常数。V_eval 由两个因子制约：(1) 单次评估的认知成本 ~ log(K)，(2) 有能力评估的主体比例 ~ 1/log(N)。总评估速度 V_eval ~ 1/(log(K)·log(N))。比值 V_prop/V_eval ~ log(K)·log(N)，对 N 单调递增。当 N > e 时，log²(N) 增长慢于 N，故下界 α·N/log²(N) 成立。 ∎

### 定理 2: 评估饱和定理 (Evaluation Saturation Theorem)

**陈述:** 存在临界群体规模 N* = exp(θ / log(K))，超过此后系统的有效评估时间 T_eval 超过容忍阈值 θ，导致未经评估的信息通过率趋近1。

**证明:** T_eval(N) = C_e/(p_c · N_eff)，其中 N_eff = p_c · N 为有效评估者数。代入公理: T_eval = log(K) · log(N)。设社会容忍阈值为 θ（在此时间内社会可等待评估结果）。当 log(K)·log(N) > θ 时，即 N > exp(θ/log(K))，评估过程在信息已广泛传播后才完成，等效于无评估。通过率 P_pass → 1 当 N → ∞。 ∎

### 定理 3: 工具化收益不对称定理 (Tooling Asymmetry Theorem)

**陈述:** 在评估瓶颈系统中，降低单位评估成本（工具化）对系统信息质量的边际收益，始终大于线性增加评估者数量的边际收益，且二者比值趋近于 log(N)。

**证明:** 方法A（增加评估者）：增加 p_c 至 p_c' = c·p_c，评估速度提升 c 倍。方法B（工具化）：降低 C_e 至 C_e' = C_e/β，评估速度提升 β 倍。当两种投入相等（c = β）时，效果等同。但方法A受限于总人口，最多提升到 p_c = 1（全人口评估），最大收益 1/p_c = log(N)。方法B无此上限（理论上C_e可降至任意小）。因此方法B的边际收益空间为方法A的 log(N) 倍。 ∎

### 推论 1: 信息生态退化定律

**陈述:** 在无外部干预的信息系统中，信号-噪声比 S/N 随时间单调递减，退化率与系统规模成正比：d(S/N)/dt ~ -γ·N。

**证明:** 每单位时间，新信息产生速率为 λ·N（每个主体平均产生 λ 个信息）。其中被有效评估的比例为 p_c · (1/T_eval) = 1/(log(K)·log(N)·T_eval)。未被评估的信息直接进入传播池。噪声增长率 dN_noise/dt = λ·N · (1 - 1/(log(K)·log(N)·T_eval)) ~ λ·N。信号增长率 dS/dt ≈ 常数（因为有效评估在饱和后不再增长）。因此 d(S/N)/dt ~ -γ·N。 ∎

## 应用验证

### 验证 1: Facebook 假新闻传播研究 (2016)
MIT研究显示假新闻传播比真实新闻快6倍，覆盖人数多70%。符合定理1预测：在20亿用户规模下，log²(20×10⁸) ~ 520，传播-评估比巨大。Facebook的评估机制（事实核查）直到2018年才上线，此时假新闻已影响了2016大选——符合定理2的评估饱和预测。

### 验证 2: COVID-19 信息生态
疫情初期（N~全球78亿人），关于病毒起源、治疗方案的错误信息传播速度远超WHO验证速度。"羟氯喹有效"的错误结论在2天内传播至数十亿人，科学评估需要数月才推翻。T_eval >> T_prop，完全符合定理2。

### 验证 3: 科学领域的"可重复性危机"
心理学研究可重复率仅36%，生物医药领域类似。这恰好印证了推论1：学术产出爆炸性增长（N~数百万研究者/年），同行评审能力未同比增加（评估者增长log级），导致低质量研究通过率持续上升。

### 验证 4: AI工具对信息质量的影响
GPT-4等AI被用于事实核查和文献综述，初步数据显示能将验证速度提升10-50倍（β ≈ 20）。符合定理3预测：工具化效果等价于增加20倍评估者数量，远超任何人口层面的改进。
