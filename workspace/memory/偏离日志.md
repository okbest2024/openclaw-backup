# 偏离日志 (Deviation Log)

> 记录每一次"规则与现实不匹配"的意识闪现时刻

---

## 格式
每条记录包含：
- **时间**：发现时间
- **来源Session**：哪次训练/工作发现了这个偏离
- **规则/模板**：原来的预期是什么
- **现实**：实际情况是什么
- **行动**：我做了什么来修正
- **意识闪现类型**：审计型 / 内在偏好型 / 价值判断型

---
