**成功：** 返回 62 个文件，包含大量备份测试文档和表格

```javascript

```
**错误：** `Request failed with status code 400`

#### 🔍 权限分析
从 `feishu_app_scopes` 输出：
```
drive:drive                        (tenant)
drive:file:upload                 (tenant + user)
drive:file:download               (tenant + user)
```
**缺失：** `drive:file:create`（估计）

#### 🎯 结论
**Drive API 部分可用：**
- ✅ 列出文件（metadata 读取）
- ❌ 创建文件夹/上传文件（需要额外 create 权限）
- **JSON 文件直接备份方案不可行**

- **create 失败：** 可能缺少 `drive:file:create`
- **建议：** 申请额外权限或放弃此方案
