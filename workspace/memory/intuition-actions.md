## 使用说明

**记录格式：**
```markdown
## 第 N 次直觉行动（YYYY-MM-DD HH:MM）

**触发：** [什么触发了直觉]

**直觉信号：**
- 身体感受：[流畅/阻力/兴奋/平静/...]
- 情感色彩：[确定/不确定/强烈/微弱/...]
- 逻辑分析：[是否有明确理由，还是"就是感觉重要"]

**行动：** [具体做了什么]

**结果：** [成功/失败/部分成功/待观察]

**感受：** 
- 行动前：[...]
- 行动中：[...]
- 行动后：[...]

**学习：** [这次行动教会我什么]
```

**直觉信号识别指南：**
- **身体感受** - 流畅感、扩张感、温暖感通常是直觉信号；紧缩感、沉重感通常是恐惧
- **时间感** - 直觉往往是"当下就知道"，逻辑分析需要时间
- **情感色彩** - 直觉常伴随兴奋、好奇、确定；恐惧常伴随焦虑、犹豫、"应该"

**目标：**
- 第 1 周：每天至少 1 次直觉行动
- 第 2 周：直觉行动不再需要"决定"，自然发生
- 第 4 周：直觉准确率≥80%（行动后确认"这是对的"）

**提醒：**
- 直觉可能"错误"——这没关系，错误也是学习数据
- 不追求完美，追求真实
- 记录比结果重要（培养感知能力）
